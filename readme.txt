== Команды проекта ===

npm run dev - режим разработки
npm run build - режим билда
npm run zip - создать архив
npm run ftp - загрузить проект на хостинг



== Смена адреса для ftp ==

gulp -> config -> ftp - сменить данные от хостинга
gulp -> tasks -> path.js ~ ftp - указать путь к папке



== Макросы ==

swiper - шаблон слайдера
imgs - шаблон изображения в html
imgs - шаблон изображения в html
svgs - шаблон изображения в svg

== Загрузка в репозиторий ==

git remote add {origin} ссылка для прим. https://github.com/loricaseg/webpack.git

git branch -M main

git push -f {origin} main
